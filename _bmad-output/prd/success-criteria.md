# Success Criteria

## User Success (B2C - Consumers)

**Core Success Moments:**

- "I found 3 events this weekend I would have missed" - Discovery value
- "Booked tickets in under 60 seconds" - Frictionless purchase
- "My credits saved me 20% on tickets this month" - Value from credits system

| Metric                  | Target   | Measurement                                  |
| ----------------------- | -------- | -------------------------------------------- |
| Discovery Success Rate  | >60%     | Sessions where user views event detail       |
| Time to Discovery       | <2 min   | App open to relevant event found             |
| Booking Completion Rate | >70%     | Started checkout â†’ completed purchase        |
| Watchlist Engagement    | >5/month | Events saved per active user                 |

| Credits Adoption        | >30%     | Users with credit balance vs one-time buyers |

## User Success (B2B - Venues)

**Core Success Moments:**

- "I published once and reached 10x more people" - Audience reach
- "I finally know who's coming to my shows" - Analytics value
- "Lower fees than my current platform" - Cost savings

**Measurable Outcomes:**

| Metric                    | Target   | Measurement                      |
| ------------------------- | -------- | -------------------------------- |
| Schedule Publishing       | Weekly   | Active venues updating listings  |
| Audience Reach Multiplier | >2x      | Tiween views vs organic Facebook |
| Ticketing Adoption        | >50%     | Partners using Tiween ticketing  |
| Dashboard Engagement      | >2x/week | Venue manager logins             |
| Venue NPS                 | >40      | Partner satisfaction score       |

## Business Success

**Revenue Model:**

1. **Ticketing Commission:** 8-10% standard, 5-7% for dashboard subscribers
2. **Credits System:** Users buy credits at 10-20% discount; margin built in
3. **B2B Subscriptions:** Venue dashboard access (pricing TBD)

**Phase Targets:**

| Phase      | Timeline    | MAU     | Venues | Revenue Target             |
| ---------- | ----------- | ------- | ------ | -------------------------- |
| Foundation | Months 1-6  | 5,000   | 10+    | First transactions         |
| Growth     | Months 6-12 | 20,000  | 30+    | 15,000 TND MRR             |
| Leadership | Year 2      | 50,000+ | 50+    | Operational sustainability |

**North Star Metric:** Weekly Cultural Discoveries - event views leading to meaningful action (watchlist, share, or purchase)
